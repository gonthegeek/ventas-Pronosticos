Medidas de Seguridad Implementadas

  • Autenticación de usuarios requerida para acceso
  • Política de Seguridad de Contenido (CSP) implementada
  • Headers de seguridad HTTP configurados
  • Base de datos Firebase con reglas de seguridad
  • Comunicación cifrada mediante HTTPS
  • No indexación por motores de búsqueda (robots.txt)
  • Divulgación responsable de vulnerabilidades

Declaración de Legitimidad

Esta aplicación es una herramienta empresarial legítima desarrollada para el seguimiento y análisis de ventas de máquinas expendedoras (vending).

No es una aplicación de phishing ni contiene contenido malicioso. Su propósito es facilitar la gestión interna de inventarios y análisis de rendimiento de ventas.

Si tiene dudas sobre la legitimidad de esta aplicación, puede contactarnos en security@gonzaloronzon.com

Divulgación Responsable

Si descubre una vulnerabilidad de seguridad en esta aplicación, le agradecemos que nos la reporte de manera responsable:

  • Envíe un correo a security@gonzaloronzon.com
  • Incluya una descripción detallada de la vulnerabilidad
  • Proporcione pasos para reproducir el problema
  • Permítanos 48 horas para responder inicialmente
  • Mantenga la confidencialidad hasta que se resuelva el problema

Nota Importante: Esta aplicación requiere autenticación y está destinada únicamente para personal autorizado. El acceso no autorizado está prohibido y puede estar sujeto a acciones legales.
